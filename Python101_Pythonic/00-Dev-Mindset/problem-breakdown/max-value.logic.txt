รับลิสต์ตัวเลข หาว่า “ตัวเลขที่มากที่สุด” คืออะไร

Input: list เป็นตัวเลข, มีจำนวนไม่แน่นอน, และอาจไม่ใช่ตัวเลขทั้งหมด

Output: เป็นตัวเลขที่มีค่ามากที่สุดใน list

Logic Steps: รับ list 
            -> เตรียมตัวแปร max_value is None
            -> วนค่าทุกตัวใน list 
            -> เช็คถ้าไม่ใช่ int  ข้าม 
            -> ถ้า max_value ยังไม่มีค่า -> ตั้งให้เป็น max_value 
            -> ถ้าค่าปัจจุบัน > max_value แทนค่า
            -> ครบทุกตัวใน list 
            -> return max_value

Edge Cases: -> list ว่าง
            -> list มี แต่ค่าที่ไม่ใช่ int ทั้งหมด return (None/0/error)
            -> อาจมีเลขติดลบ

Order of Checks: -> รับ list
                 -> ถ้า list ว่าง -> จบ
                 -> วนดูที่ละค่า
                 -> เช็คว่าเป็น int ไหม ใช่ทำ , ไม่ข้าม
                 -> ตั้งค่าตัวแรกให้เป็นค่ามากสุด
                 -> เปรียบเทียบกับตัวถัดไป
                 -> ถ้าค่าใดมากว่าตัวแรก ให้ค่านั้นแทนตัวแรก 
                 -> ครบทุกตัวใน list 
                 -> return ค่ามากสุดใน list
