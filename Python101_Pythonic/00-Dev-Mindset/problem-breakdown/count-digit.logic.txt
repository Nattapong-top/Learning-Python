1️⃣ Problem Solving Mindset (หัวใจที่สุด)

ทุกโจทย์ต้องตอบ 5 คำถามนี้ให้ได้ก่อนเขียนโค้ด (ที่ป๋าทำอยู่ = มาถูกทางแล้ว)

Input: รับอะไรเข้ามา
- รับ row_list = [2,34,5,6,5,5345,444]
- เลขเป้าหมาย d = 5

Output: ต้องคืนอะไร
- ตัวเลข จำนวนที่นับ เลขโดด

Logic: คิดยังไงให้ได้คำตอบ
- ตั้ง count = 0
- วงวนรอบนอก i ทุกค่าใน list มาเช็ค
- while i > 0
-   ถ้า i % 10 == d
-       count + 1
-   i // 10
- จบวงวน while และ วงวนรอบนอก
- retrun count

- นำ i เข้าสมการ ถ้า == d count + 1
