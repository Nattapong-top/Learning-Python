
1️⃣ Problem Solving Mindset (หัวใจที่สุด)

ทุกโจทย์ต้องตอบ 5 คำถามนี้ให้ได้ก่อนเขียนโค้ด (ที่ป๋าทำอยู่ = มาถูกทางแล้ว)

Input: รับอะไรเข้ามา
- รับ list_a = ค่า x ของจุดทั้งหมด
- รับ list_b = ค่า y ของจุดทั้งหมด

Output: ต้องคืนอะไร
- ตัวเลขที่ เปรียบเทียบแล้ว มีค่าน้อยที่สุด
- อาจจะมีเลขคู่ที่น้อยสุดด้วย

Logic: คิดยังไงให้ได้คำตอบ
- ตั้งตัวแปร distance 0
- ตั้งตัวแปร min_distance none
- เช็ค list ก่อนว่างไหม ว่าง return 0
- เช็คว่า มีค่าใน len(list_a) == len(list_b) เท่ากันไหม
- list น้อยกว่า 2 ทำไม่ได้ return 0
- วงวนรอบนอก i index ตามจำนวน list 
- วงวนรอบใน j index ตาจำนวน list
- วน i < j
- ตั้งค่าตัวแปร แทนค่าแต่ละจุด
- x1 = list_a[i]
- y1 = list_b[i]
- x2 = list_a[j]
- y2 = list_b[j]

- นำ x1, y1, x2, y2 ไปเข้าสูตร
- เก็บไว้ในตัวแปร distance 
- if distance < min_distance
- min_distance = distance
- x = x1, y1
- y = x2, y2
- จบ วงวน

- return x, y, min_distance


Condition / Loop: ตัดสินใจหรือวนตรงไหน
- วงวนรอบนอก i index ตามจำนวน list 
- วงวนรอบใน j index ตาจำนวน list
- นำไปเข้าสูตร เอาค่าทีได้ไปเปรียบเทียบ
Edge Case: เคสแปลก ๆ มีไหม
list ว่าง
list ไม่เท่ากัน
list มีแคจัดเดียว
มีจุดซ้ำตำแหน่งเดียวกัน distance = 0
ค่าเป็น float / int ปนกัน
