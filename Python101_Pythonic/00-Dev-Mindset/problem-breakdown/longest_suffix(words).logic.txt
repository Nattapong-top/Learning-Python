1️⃣ Problem Solving Mindset (หัวใจที่สุด)

ทุกโจทย์ต้องตอบ 5 คำถามนี้ให้ได้ก่อนเขียนโค้ด (ที่ป๋าทำอยู่ = มาถูกทางแล้ว)

Input: รับอะไรเข้ามา
- ข้อมูลเป็น list ข้อความ str 

Output: ต้องคืนอะไร
- ข้อความต่อท้ายเหมือนกันยาวที่สุด 

Logic: คิดยังไงให้ได้คำตอบ
- base ตัวแปรเริ่มต้น = list[0][::-1] กลับด้านข้อความ

สำหรับทุกindex i ใน เริ่ม 1 ถึง len(list):
    rev = กลับด้านข้อความทั้งหมด list[i][::-1]
    suffix = '' เก็บอักษรเหมือนกัน
    len_min = เก็บจำนวนอักษรที่น้อยสุด
    สำหรับทุก index j ใน len_min:
        ถ้า base[j] == rev[j]:
            suffix += base[i] เพิ่มอักษรเข้าไป
        ถ้าไม่เข้ากัน ให้หยุด
    base = suffix 
ส่งค่าคืน base[::-1]

Condition / Loop: ตัดสินใจหรือวนตรงไหน
- วงวนรอบนอก เอาไว้ นำค่าออกมา ที่ละชุด
- วงวนใน นำค่าตัวอักษาออกมาเปรียบเทียวกัน

Edge Case: เคสแปลก ๆ มีไหม
list ว่าง อักษรพิเศษ ชุดข้อความไม่เท่ากัน
